# create-web-app-cli
CLIでweb開発の様々な雛形を生成できるツール  

現在の段階では、  
Next.js(App Router), React(vite), Vue.js(vite), Tailwindcss, shadcn/ui, Storybook, ESLint/Prettier, Typescript, に対応してます。  

# 使用方法
①グローバルインストール  
```git clone https://github.com/Crazy-ontyan/create-web-app-cli.git```  
```cd create-web-app-cli```  
```npm install```  
```npm link```  
インストール後は任意のディレクトリで使えます:  
```npm create-web-app-cli```  

②クローンせず直接実行  
```npx github:Crazy-ontyan/create-web-app-cli```  

# アンインストール
```npm unlink create-web-app-cli```
